# OS

Very small Open-Source OS with a DOS shell + desktop tools using HTML + Node.js.

## Run

```bash
node server.js
```

Open: `http://localhost:3000`

## Desktop tools

- **Commands** (opens DOS shell window at the bottom-right)
- **Image Editor** (simple paint + save PNG)
- **Quit** (halts system, restart available)

## Commands

- `help`
- `cls` / `clear`
- `echo <text>`
- `date`
- `time`
- `ver`
- `ls` / `dir`
- `cat <file>`
- `pwd`
- `whoami`
- `uptime`
- `touch <file>`
- `write <file> <text>`
- `del <file>` / `rm <file>`
- `rename <old> <new>`
- `open <commands|image>`
- `quit`
